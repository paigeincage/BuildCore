Native app · Round 1 · Critical workflow

Upload → Auto-sort → Send

The product's core moment of value. A user uploads a punch list (PDF, photo, doc); The Condenser extracts every item, auto-sorts by trade, lets the user edit, then sends the trade-specific list to the right contact. Three screens — designed to feel like one continuous flow.

⚠ Sample data only.
Every address, file name, trade, and contact below is placeholder. At runtime, content is driven by:

user profile → communities → sites → trades → contacts.
01 · Intake

Upload files

Drop PDFs, docs, spreadsheets, or jobsite photos. The drop zone is also the iOS file/photo picker on tap. Files queue up with type + size + remove.

02 · Review

Auto-sorted by trade

Items return grouped by trade, with priority dots, location chips, and source-file captions. Tap to edit any line. Items marked "repaired" auto-deselect.

03 · Send sheet

Bottom sheet · iOS

Trade-by-trade. Auto-matches a contact from the user's address book by trade preference. Two-up primary actions: Email · Text. Disabled if the contact's channel is invalid.
