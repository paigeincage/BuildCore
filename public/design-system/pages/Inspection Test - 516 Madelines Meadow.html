  • Severity from a TREC inspection is mostly "Minor / Maintenance" — that maps to "normal" priority. Only 2 items hit "Marginal Concerns" (elevated). The live app's Claude-based extraction would weight these differently.
  • Location detection is heuristic — many items show empty location. The real extractor (using Claude) can pull location from the surrounding sentences better.
